トレーサ（xonsh.tracer）¶

xonshトレーサを実装します。

class xonsh.tracer.TracerType[source] ¶

すべてのトレース状態を追跡するxonshトレーサオブジェクトを表します。これはシングルトンです。

color_output（usecolor ）[ソース] ¶

トレーサ出力を着色するかどうかを指定します。
