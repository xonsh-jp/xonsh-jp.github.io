カラーツール（xonsh.color_tools）¶

xonshでの色の操作のためのツール。

これには、RGB 16進コードとxterm-256カラーコード間の値の変換が含まれます。このファイルの一部は、もともとはMicah Elliott http://MicahElliott.comの Copyright（C）2011 Micah Elliott からフォークされていました 。全著作権所有。WTFPL http://sam.zoy.org/wtfpl/

xonsh.color_tools.rgb2short（rgb ）¶

与えられたRGB値に最も近いANSI 256近似を求める。

>>> rgb2short('123456')
('23', '005f5f')
>>> rgb2short('ffffff')
('231', 'ffffff')
>>> rgb2short('0DADD6') # vimeo logo
('38', '00afd7')
パラメーター：
rgb ： RGB値を表す16進数コード、例えば 'abcdef'
戻り値：
xtermと互換性のある0〜255の文字列。
