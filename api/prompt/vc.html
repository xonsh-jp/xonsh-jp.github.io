バージョン管理ブランチ情報プロンプトformatter（xonsh.prompt.vc）¶

簡単なバージョン管理ブランチのためのプロンプト・フォーマッタ

xonsh.prompt.vc.branch_bg_color（）[ソース] ¶

現在のブランチが汚れている場合は赤、汚れが判別できない場合は黄、汚れている場合は緑に戻ります。これらは背景色です。

xonsh.prompt.vc.branch_color（）[ソース] ¶

現在のブランチが汚れている場合は赤、汚れが判別できない場合は黄、汚れている場合は緑に戻ります。これらは、フォアグラウンドのための大胆で強い色です。

xonsh.prompt.vc.current_branch（）[ソース] ¶

現在の作業ディレクトリのブランチを取得します。cwdがリポジトリでない場合は、空の文字列を返します。これは現在gitとhgでしか動作せず、将来拡張されるべきです。タイムアウトが発生した場合、文字列 '<branch-timeout>'が返されます。

xonsh.prompt.vc.dirty_working_directory（）[ソース] ¶

バージョン管理リポジトリ内にコミットされていないファイルがあるかどうかを示すブール値を返します。これを判別できない場合は、Noneを返します。現在、gitとhgをサポートしています。

xonsh.prompt.vc.get_git_branch（）[ソース] ¶

現在のgitブランチを探します。これが決定できない場合（タイムアウト、git repoなどではない）、Noneを返します。

xonsh.prompt.vc.get_hg_branch（root = None ）[ソース] ¶

現在のディレクトリの水銀分岐を取得し、レポまたはサブプロセスにない場合はNoneを返します。タイムアウトした場合はタイムアウトします。

xonsh.prompt.vc.git_dirty_working_directory（include_untracked = False ）[ソース] ¶

gitディレクトリがdirtyかどうかを返します。これを判別できなかった場合（タイムアウト、ファイルが見つからないなど）、Noneを返します。

xonsh.prompt.vc.hg_dirty_working_directory（）[ソース] ¶

水銀作業ディレクトリが汚れているかどうかを計算します。これが判別できない場合は、Noneが戻されます。
