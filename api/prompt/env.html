Virtualenvプロンプトformatter（xonsh.prompt.env）¶

virtualenvなどのためのプロンプト・フォーマッタ

xonsh.prompt.env.env_name（pre_chars = '（'、post_chars = '）' ）[ソース] ¶

現在の環境名を$ VIRTUAL_ENVまたは$ CONDA_DEFAULT_ENVから抽出します

xonsh.prompt.env.vte_new_tab_cwd（）[ソース] ¶

これにより、VTE端末にホスト名とpwdを伝えるエスケープシーケンスが出力されます。これはほとんどの場合は必要ないはずですが、始動時に環境からPWDを読み取らないLinux端末の場合があります。これは文字列を返さず、エスケープシーケンスを単に出力してstdoutに直接フラッシュすることに注意してください。
