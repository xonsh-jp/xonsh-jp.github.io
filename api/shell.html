メインシェルコマンドプロンプト（xonsh.shell）¶

xonshシェル

クラスxonsh.shell.Shell（execer、ctx = None、shell_type = None、** kwargs ）[ソース] ¶

主なxonshシェル。

実行環境を初期化し、prompt_toolkitまたはreadline版のシェルを使用するかどうかを決定します。

パラメーター：
実行者 ： 実行者

xonshコードを実行できるexecutorインスタンス。

ctx ： マッピング、オプション

シェルの実行コンテキスト（例えば、グローバル名前空間）。存在しない場合、これはrcファイルをロードすることによって計算されます。Noneでない場合、これは追加のコンテキストが計算されず、これが直接使用されます。

shell_type ： str、オプション

'readline'、 'prompt_toolkit1'、 'random'など、起動するシェルのタイプ。
xonsh.shell.transform_command（src、show_diff = True ）[ソース] ¶

プリコマンドハンドルを実行した結果を返します。
