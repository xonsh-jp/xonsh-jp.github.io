ファイルシステムパスコンプリート（xonsh.completers.path）¶

xonsh.completers.path.cd_in_command（行）[ソース] ¶

"cd"が行のトークンであればTrueを返し、そうでなければFalseを返します。

xonsh.completers.path.complete_dir（接頭辞、行、開始、終了、ctx、cdpath = False ）[ソース] ¶
xonsh.completers.path.complete_path（接頭辞、行、開始、終了、ctx、cdpath = True、filtfunc =なし）[ソース] ¶

パス名に基づいて完了します。

xonsh.completers.path.subsequence_match（ref、typed、csc ）[ソース] ¶

型付きがrefのサブシーケンスであるかどうかを検出します。

True文字がどこにあるかにかかわらず、文字typedが（順序通りに）表示さ れているかどうかを返します。場合である 、の場合を無視して。refrefcscFalsereftyped

「サブシーケンス」のパス補完（例えば、~/u/ro展開 ~/lou/carcohl）で使用されます。
