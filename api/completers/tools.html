完了ツール（xonsh.completers.tools）¶

Xonshコンプリータツール。

xonsh.completers.tools.get_filter_function（）[ソース] ¶

$ CASE_SENSITIVE_COMPLETIONSの有効性を考慮して、補完に適切なフィルタリング関数を返します。

xonsh.completers.tools.justify（s、max_length、left_pad = 0 ）[ソース] ¶

各行がmax_length文字以下の長さになるように文字列sを再ラップします。最初の行はすべてleft_padという文字列で埋められます。
