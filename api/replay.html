リプレイの履歴（xonsh.replay）¶

xonshの履歴ファイルを再生するツール。

class xonsh.replay.Replayer（f、reopen = True ）[ソース] ¶

xonshの履歴ファイルを再生します。

パラメーター：
f ： ファイルハンドルまたは文字列

xonsh履歴ファイルのパス。

再オープン ： bool、オプション

新しいファイルハンドルをロードごとにオープンするかどうかは、LazyJSONクラスに直接渡されます。
replay（merge_envs =（ '再生'、'ネイティブ'）、ターゲット=なし）[ソース] ¶

指定された履歴を再生し、コードが実行された履歴オブジェクトを返します。

パラメーター：
merge_env ： strまたはMappingsのタプル、オプション

優先順位の高い順に環境をマージする方法について説明します。使用可能な文字列は「再生」と「ネイティブ」です。「再生」のenvは再生中の履歴ファイルから来ます。'ネイティブ' envは、xonshのこのインスタンスが起動されたときのものです。文字列の代わりに、dictやその他のマッピングを渡すこともできます。デフォルトは（ '再生'、 'ネイティブ'）です。

target ： str、オプション

新しい履歴ファイルのパス。
